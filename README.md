# 🐾 Vera Pet Shop

Proyecto realizado como trabajo integrador para el curso de Front End JavaScript de Talento Tech 2026.

## Tecnologías

Se trata de una tienda online de productos para mascotas desarrollada utilizando HTML, CSS y JavaScript modular, con almacenamiento del carrito mediante LocalStorage.

### Funcionalidades

- Catálogo de productos cargado dinámicamente desde un archivo JSON
- Carrito de compras con persistencia en localStorage
- Contador de productos en el carrito visible en todas las páginas
- Formulario de contacto integrado con Formspree
- Diseño responsive para dispositivos móviles y escritorio
